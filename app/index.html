Country Select

  • Source

Demo

All you need to do is drop the <select> item into the HTML.

The plugin will do the rest.

<select name="country" class="flags">
  <option class="flag flag-{countrycode}">Country Name</option>
  ...
</select>

Here's an example of African countries:

Requirements

  • Include the plugin CSS.
  • Include the plugin JS after jQuery.
  • Add the proper classes to your <select> elements

Notes:

The {countrycode} variable that you attach to the option items are based on the following spec: ISO 3166-1 alpha-2.
